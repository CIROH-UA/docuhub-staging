NGIAB-CloudInfra

NGIAB version 1.5.0 is available now. This update restricts searches for realization.json to the config folder. Additionally, the LSTM model has been integrated into the NGIAB image, enabling AI-powered hydrologic prediction.

NGIAB-HPCInfra

NGIAB-HPCInfra version 1.1.0 is available now. This update brings parity with NGIAB-CloudInfra's recent changes, including full support for LSTM-based models.

ngen-datastream

ngen-datastream is now known as the NextGen Research DataStream (NRDS)! We hope to continue using this platform to advance the boundaries of continental-scale modeling.

Release v1.1.0 recently released! Major enhancements include implementation of medium range ensemble members in DataStreamCLI and NRDS infrastructure, enhanced metadata and CI/CD with semantic versioning implementation, development towards automated testing and production deployment, the DataStreamCLI skip feature, increased test coverage, and documentation updates.
